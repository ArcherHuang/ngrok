# ngrok

## 【 Overview 】  
         
## 【 Setup ngrok 】

* Download ngrok
   * [Mac OS](https://bin.equinox.io/c/4VmDzA7iaHb/ngrok-stable-darwin-amd64.zip)
   * [Windows](https://bin.equinox.io/c/4VmDzA7iaHb/ngrok-stable-windows-amd64.zip)

* Unzip
   * Mac OS
   
     ```
     unzip /path/to/ngrok.zip
     ```

## 【 Run it 】

* Python Flask

     ```
     pip install flask
     python crud.py
     ```

* ngrok

     ```
     ./ngrok http 5000
     ```
